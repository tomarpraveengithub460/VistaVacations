# VistaVacations
Full Stack Project

STEPS :
1. Open VS Code
2. Run : git clone <-----link------>
3. npm install
4. Add Your credentials of Mongo Atlas in .env file

6. To run Project on local system -> Add your current IP Address to Mongo Atlas (if using)
7. Run : npx nodemon app.js
